About Me

I am a front-end developer who believes that web design should be accessible, high-performing, and beautiful for everyone. By combining clean coding standards with creative layout structures, I build websites that adapt to any screen size and respect user preferences.

This portfolio is built entirely from scratch, utilizing fluid layouts, media queries, CSS Custom Variables, custom grid structures, and interactive JavaScript filters.

Core Competencies

  • HTML5 & Accessibility (WCAG)
  • CSS3, Flexbox & Grid
  • Responsive Design (Mobile-First)
  • JavaScript & DOM Manipulation
