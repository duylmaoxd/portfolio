2. Is the site responsive?

The site is designed mobile-first and uses pure CSS Grid and Flexbox layouts. Resize your browser window to see three distinct layout structures:

  • Mobile Viewport (< 768px): 1-column layouts, off-canvas sliding mobile menu.
  • Tablet Viewport (≥ 768px): 2-column flex rows, navigation returns inline.
  • Desktop Viewport (≥ 1024px): 3-column grid system for project showcase cards.
10 / 10

3. Is the site well-styled?

Styled from scratch using standard CSS3, a custom color palette, smooth transition matrices, keyframe-based SVG illustrations, and Outfit & Space Grotesk Google Fonts. Custom keyboard focus styles ensure high accessibility contrast.

10 / 10

4. Are images effective?

All images are light-weight SVGs with descriptive `alt` tags and explicit dimensions to prevent cumulative layout shift (CLS), behaving responsively within custom container frames.

40 / 40

5. Enhancements (The Four Extras)

Extra 1: Light/Dark Mode Switcher

A persistent JavaScript theme switcher that modifies document parameters, reads preferences from localStorage, and respects system settings via prefers-color-scheme.

Extra 2: Accessible Image Carousel

An custom image slider (see below) featuring manual controls, indicator dots, keyboard navigators, and a Play/Pause button. Automatic starting is disabled by default as requested by the rubric.

Extra 3: Sticky Footer Layout

Utilizes modern Flexbox layout architecture to ensure the footer remains consistently anchored to the bottom of the viewport even on screens with minimal content height.

Extra 4: Dynamic Project Filter

JavaScript DOM script that filters project items by category dynamically on button clicks, applying smooth CSS opacity transitions to coordinate visual feedback.

Featured Landscapes Carousel

An interactive, screen-reader friendly image carousel demonstrating dynamic slide control. Use the arrow keys, indicators, or controls to navigate. Autoplay is paused by default.
